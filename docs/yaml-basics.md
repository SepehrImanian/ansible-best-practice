```yaml
name: sepehr

age: 12
```

Array / List :
```yaml
courses:
- linux
- docker
- kubernetes
```

Dictionary / Map:
```yaml
grade:
    docker: 20
    linux: 18
    kubernetes: 20
```

```yaml
Fruits:
    - Banana:
        Calories: 105
        Fat: 0.4 g
        Carbs: 0.4 g
    - Grape:
        Calories: 62
        Fat: 0.3 g
        Carbs: 17 g
```

Dictionary = Unordered \
List = Ordered \
